# Sarcasm & Sentiment Classifier for E-commerce Conversations

## Overview
This capstone project builds a transformer-based model (DeBERTa v3) to detect sarcasm in mult-turn customer conversations.

## Features
- Fine-tuned DeBERTa sarcasm classifier
- 2500 multi-turn ecommerce conversations dataset
- FastAPI backend (`/predict`)
- Streamlit UI for interactive predictions
